### 2. 分红落袋逻辑优化 (`src/quant_core/engines/backtest.py`)

**收益计算逻辑重构** (`_calc_return_breakdown` 方法)：

```python
# 从每日资金曲线提取分红金额
for i in range(1, len(self.equity_curve)):
    prev = self.equity_curve[i - 1]
    curr = self.equity_curve[i]
    
    # 基于累计净值和单位净值的差异计算分红
    nav_based_value = prev['shares'] * curr['nav'] if prev['shares'] > 0 else 0
    
    if curr_value > nav_based_value and prev['shares'] > 0:
        daily_dividend = curr_value - nav_based_value
        total_dividend_reinvest += daily_dividend
        total_dividend_cash += daily_dividend

# 分红再投：分红自动再投资，享受复利
dividend_reinvest = (total_dividend_reinvest / initial_capital) * 100
```

### 分红落袋 vs 再投对比

| 模式 | 逻辑 | 收益率 |
|------|------|--------|
| **分红再投** | 分红自动买入更多份额，享受后续复利增长 | 较高（含复利）|
| **分红落袋** | 分红作为现金持有，下次买入时连同本金使用 | 较低（无复利）|
