## Bug 修复：akshare 数据获取失败 (17:45)

### 问题
`ak.index_zh_a_hist()` 接口返回 `Connection aborted` 错误，导致沪深300指数数据获取失败。

### 原因
`index_zh_a_hist` 接口依赖的网络连接不稳定，可能被服务器拒绝。

### 解决方案
在 `akshare_provider.py` 中实现**双接口容错机制**：
1. 主接口：`index_zh_a_hist`（支持日期范围参数）
2. 备用接口：`stock_zh_index_daily`（获取全部历史数据，本地过滤）

```python
try:
    df = ak.index_zh_a_hist(...)
except Exception as e1:
    logger.warning(f"主接口失败，尝试备用接口...")
    symbol = f"{market}{code}"
    df = ak.stock_zh_index_daily(symbol=symbol)
```

### 修复文件
- `src/quant_core/providers/akshare_provider.py`
