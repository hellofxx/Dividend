---

## 1. 产品概述

### 1.1 核心价值主张

> **"让投资分析像看图表一样简单，让策略验证像玩游戏一样直观"**

量化工作室提供两种互补的分析模式：

| 模式 | 核心价值 | 适用场景 |
|------|---------|---------|
| **判断模式** | 实时技术分析、多因子信号判定 | 当下该不该买/卖 |
| **回测模式** | 历史策略验证、基准对比分析 | 这个策略长期有效吗 |

### 1.2 双模式架构

```
                    ┌─────────────────────────────────┐
                    │     量化工作室 QuantStudio      │
                    │         quant_studio.py         │
                    └───────────────┬─────────────────┘
                                    │
              ┌─────────────────────┴─────────────────────┐
              │                                           │
    ┌─────────▼─────────┐                     ┌─────────▼─────────┐
    │    判断模式        │                     │    回测模式        │
    │  (Judgment Mode)  │                     │  (Backtest Mode)  │
    ├───────────────────┤                     ├───────────────────┤
    │ • 技术分析图表     │                     │ • 量化回测引擎     │
    │ • 多因子信号判定   │                     │ • 基准对比分析     │
    │ • 实时行情展示     │                     │ • 收益分解验证     │
    │ • 6大技术指标     │                     │ • 超额收益计算     │
    └───────────────────┘                     └───────────────────┘
```

### 1.3 运行环境

- **平台限定**：Windows 桌面环境，VSCode 集成终端
- **Conda 环境**：study 环境，Python 3.9+
- **核心依赖**：pandas, numpy, matplotlib, akshare, pyyaml
- **性能优化依赖**：numba (0.60.0+), plotly (5.24.0+)

### 1.4 目标用户

| 用户类型 | 核心需求 | 推荐模式 |
|---------|---------|---------|
| ETF投资者 | 判断当前是否该买入/卖出 | 判断模式 |

### 2.2 判断模式功能

#### 2.2.1 技术分析系统

| 功能点 | 描述 | 验收标准 |
|--------|------|---------|
| ETF数据获取 | 调用公共数据模块获取ETF数据 | 数据获取成功，包含最近6个月历史数据 |
| 技术指标计算 | 计算BIAS、MACD、RSI、KDJ、布林带等指标 | 计算结果准确，与标准公式一致 |
| 图表展示系统 | 展示6个专业技术分析图表 | 图表布局合理，交互流畅 |
| 买入信号判定 | 基于多因子综合判定买入信号 | 信号判定准确，依据清晰 |

#### 2.2.2 图表展示系统（5个专业技术分析图表）

| 图表 | 内容 | 技术指标 |
|------|------|---------|
| K线走势图 | ETF价格K线走势，支持蜡烛图/折线图切换，叠加BIAS，以数据起始点为基准值1，包含年线指标 | BIAS乖离率、MA250 |
| MACD指标图 | DIF线、DEA线和MACD柱状线 | MACD(12,26,9) |
| RSI指标图 | RSI指标，支持自定义周期 | RSI(14) |
| KDJ指标图 | K线、D线和J线 | KDJ(9,3,3) |
| 布林带指标图 | 中轨、上轨和下轨 | Bollinger(20,2) |

#### 2.2.3 买入信号判定系统

| 功能点 | 描述 | 判定规则 |
|--------|------|---------|
| 多因子综合判定 | 基于BIAS、MACD、RSI、KDJ、布林带等因子综合判定 | 3个以上因子看多则买入 |
| 信号判定规则 | 建立明确的信号判定规则库 | 见2.2.4节 |
| 判定依据说明 | 信号结果附带判定依据说明 | 展示各指标状态 |

#### 2.2.4 判断模式信号规则

| 指标 | 买入因子 | 卖出因子 |
|------|---------|---------|
| BIAS | bias < 0 (低于年线) | bias > 10 (严重超买) |
| MACD | 金叉 (DIF上穿DEA) | 死叉 (DIF下穿DEA) |
| RSI | RSI < 30 (超卖) | RSI > 70 (超买) |
| KDJ | KDJ < 20 或 金叉 | KDJ > 80 或 死叉 |
| 布林带 | 价格触及下轨 | 价格触及上轨 |

**综合判定规则**：
- 买入信号：买入因子数 >= 3
- 卖出信号：卖出因子数 >= 3
- 观望信号：其他情况

### 2.3 回测模式功能

#### 2.3.1 核心回测引擎

| 功能点 | 描述 | 验收标准 |
|--------|------|---------|
| 策略规则配置 | 完全可配置的规则链，支持分批交易 | 支持自定义买卖阈值 |
| 分红模式 | 支持 reinvest(再投) 和 cash(落袋) 两种模式 | 分红逻辑正确 |
| 最小单位约束 | 可用资金不足以买入0.01份时放弃 | 符合业务规则 |
| 防重入约束 | 同一交易日相同触发条件仅执行一次 | 避免重复交易 |
| 期末结算 | 回测结束日仍持仓按最后净值计算 | 资金曲线完整 |
| 图片保存功能 | 默认不自动保存图片，仅通过明确的保存按钮触发 | 点击保存按钮时才保存图片 |
| 批量保存功能 | 支持批量保存所有回测结果图片 | 点击"保存所有图片"按钮时保存全部图片 |

#### 2.3.2 基准对比模块（独立）

| 基准 | 描述 | 实现方式 |
|------|------|---------|
| 一次性买入 | 起始日全仓买入持有至结束日 | 基准A |
| 定期定额(DCA) | 每自然月第一个交易日固定投入10000元 | 基准B，月金额可配置 |
| 沪深300 | 同期沪深300指数收益 | 基准C |
| 上证指数 | 同期上证指数收益 | 基准D |

#### 2.3.3 策略对比指标体系

| 指标分类 | 具体指标 | 业务含义 |
|---------|---------|---------|
| **收益类** | 累计收益率 | 整个回测期内的总收益 |
| | 年化收益率 | 年度复合收益率 |
| | 净值收益率 | 仅净值变动收益 |
| | 分红收益(再投/落袋) | 分红模式收益拆解 |
| **对标类** | 沪深300同期收益率 | 大盘基准表现 |
| | **超额收益率** | 主策略 - 沪深300 |
| **风险类** | 最大回撤 | 最极端亏损幅度 |
| **风险调整收益** | 夏普比率 | 单位风险超额回报 |
| | 卡玛比率 | 单位回撤风险回报 |
| **交易特征** | 胜率 | 盈利交易占比 |
| | 盈亏比 | 平均盈利/平均亏损 |
